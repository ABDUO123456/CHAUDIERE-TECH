INSTALLATION PROTOCOL

DANGER: GAS CONNECTION MUST BE CARRIED OUT BY QUALIFIED PERSONNEL ONLY. LEAK TEST IS MANDATORY BEFORE IGNITION.
01
ELECTRICAL WIRING

Connect to a 230V~50Hz supply. Ensure correct Phase-Neutral polarity for the ionization flame detection to function.

Ground connection is mandatory (Yellow/Green cable).
02
GAS SUPPLY

Use 22mm copper pipe if the distance from the meter exceeds 3 meters. Minimum gas inlet pressure: 20 mbar (G20).

Install an external gas shut-off valve near the boiler.
03
FLUE SYSTEM

Standard coaxial pipe 60/100mm. Maximum equivalent length: 6 meters (including one 90° bend).

Ensure a 3% slope downwards towards the boiler for condensation.
